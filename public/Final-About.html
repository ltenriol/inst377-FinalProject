Movie Matchmaker

Matching you with the movies you'll love!

About

When you live in a world where there are hundreds of thousands of available movies and shows to watch, finding the right one may be overwhelming. Not to mention, the time to scroll through all the options will take away that precious time you could be spending to watch a film or movie. This is where Movie Matchmaker comes in. Movie Matchmaker is a web application that is created to help users discover new movies and shows with ease; all while making the process fun and engaging!

How It Works

Movie Matchmaker uses real time data from the TMDB API to deliver accurate and up to date information about movies and shows. The matchmaking system is designed to be intuitive and user-friendly, and offer an interactive experience. Users will use an engaging matchmaking system that are mainly known to be used in dating applications such as Tinder and Bumble. This will allow them to scroll through many options effortlesly. If a user accepts a movie or show, it will be added to their watchlist. If a user rejects a movie or show, it will be discarded.
